# G2 Convert

MacOS native image converter for Even Realities G2 smart glasses display.

Converts any image to the G2's 4-bit greyscale format with Floyd-Steinberg dithering, interactive crop locked to the output aspect ratio, and real-time inversion. Packaged as a double-clickable `.app` bundle.

## Quick Start

```bash
./build.sh                    # full build → .app bundle with versioning
open G2Convert.app            # launch the GUI
.build/release/G2Convert --version   # print version metadata
```

Drop an image or click Browse. Pick a preset (288×144 single eye, 576×288 dual eye, or custom). Save the result as PNG.

For batch/script usage, use the lighter Python CLI in `../../scripts/g2-convert`.

## Build

```bash
./build.sh
```

What it does:
1. Reads `VERSION` (semver, e.g. `1.0.0`)
2. Increments `.build-number` (1, 2, 3…)
3. Captures current git hash (`git rev-parse --short HEAD`)
4. Generates `Sources/Version.swift` with all metadata
5. Updates `G2Convert.app/Contents/Info.plist` (version, build, copyright)
6. Compiles release binary (`swift build -c release`)
7. Copies binary into the `.app` bundle

## Usage

### GUI

| Feature | How |
|---------|-----|
| Load image | Drag & drop onto left panel, or click Browse |
| Select preset | 288×144 / 576×288 / Custom (W×H fields) |
| Crop | Enable Crop toggle → drag the overlay rectangle → adjust scale with slider |
| Invert | Toggle Invert (instant, no reprocess — works on the final dithered output) |
| 1:1 preview | Toggle 1:1 to see pixel-accurate size |
| Save | Cmd+S or Save As… → PNG |

### Crop details

- The crop rectangle is **aspect-ratio-locked to the selected preset** (e.g. 2:1 for single eye)
- Drag anywhere on the source image to reposition
- Scale slider controls crop size (10%–100%)
- At 100% scale, the crop takes the maximum possible area that preserves the output ratio
- The overlay accurately reflects what will be cropped — no visual mismatch

### Invert

- Inversion is computed **on the fly** from the cached dithered image
- Toggling invert does NOT trigger a full pipeline reprocess — it's instant
- Changing crop/preset preserves the invert state
- Loading a new image resets invert to off
