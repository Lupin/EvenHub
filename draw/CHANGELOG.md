# DRAW Changelog

## 1.1.6 — 2026-06-16

FIX
- Store listing now shows description, author, and permissions

## 1.1.5 — 2026-06-16

FIX
- Drawings now survive app kill on iPhone
- Auto-save while drawing (every 500ms)
- Save on app close so nothing is lost

## 1.1.4 — 2026-06-16

FIX
- Saved drawings no longer overwritten when external drawings exist

## 1.1.3 — 2026-06-16

FIX
- Drawings survive app updates without any special code
- Startup pattern (Vasarely) renders correctly on first launch

## 1.1.2 — 2026-06-16

FIX
- Drawings no longer disappear after updating the app
- Startup pattern only appears when the gallery is truly empty

## 1.1.1 — 2026-06-16

FIX
- Mini-map responds to touch on iPhone/iPad
- Drawing on the main canvas no longer blocked by the mini-map

## 1.1.0 — 2026-06-15

ADD
- Zoom Edit Mode: precision drawing with mini-map and 7×7 zoomed grid
- Tap mini-map to move zoom area
- 17 brush sets: Blocks, Triangles, Shapes, Box 1-5, Blocks 2-3, Mix, Alphabet, Numbers, Symbols, Arrows, Hangul, Katakana
- Edit button in gallery: reload a saved drawing back to the canvas
- Columns and Rows dropdowns for quick grid resizing
- Compact toolbar: Grid, Columns+Rows, Eraser, Clear, Zoom on one row

## 1.0.0 — 2026-06-15

ADD
- Draw mode: 28×7 resizable grid, 11 brush sets, Eraser, Clear, Save, Push to glasses
- Text mode: 4-character 5×5 block letters, Random toggle, Save
- Gallery: Display, Share, Rename, Delete, Export All
- Glasses navigation: swipe → gallery, tap → next drawing, double-tap → back, double-tap again → exit
- Dark and Light theme
- "I♥ Vasarely" default drawing
- Help page with full tutorial
